# Biblioteca en Java

Aplicación desarrollada en **Java** que permite gestionar el catálogo de una biblioteca, aplicando principios de programación orientada a objetos y características avanzadas del lenguaje.

## Descripción del proyecto

La aplicación permite gestionar una biblioteca con su catálogo de libros, ofreciendo funcionalidades como agregar, eliminar, buscar y mostrar libros.  

Se utilizan conceptos como herencia, clases anidadas, manejo de excepciones personalizadas y trabajo con ficheros.

## Funcionalidades

- Creación de una biblioteca con nombre y director  
- Construcción de un catálogo con capacidad limitada  
- Agregar libros al catálogo (si hay espacio y no están repetidos)  
- Eliminar libros del catálogo  
- Buscar libros por ISBN  
- Mostrar todos los libros disponibles  
- Consultar número total de libros en el catálogo  
- Exportar los libros a un fichero (`libros.obj`)  
- Manejo de excepciones personalizadas  
- Control de errores cuando el catálogo no existe  

## Tecnologías utilizadas

- Lenguaje: **Java**  
- Entorno de desarrollo: **IntelliJ IDEA**  

## Objetivos de aprendizaje

- Programación Orientada a Objetos (POO)  
- Uso de clases, herencia y abstracción  
- Manejo de excepciones  
- Uso de ficheros en Java  
- Estructuración de aplicaciones  
- Control de flujo mediante condiciones
